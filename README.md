# gutenmorgen

Random Guten-Morgen-Generator for the command line.

## Usage

Run the CLI without arguments:

```sh
gutenmorgen
```

Or pass a name:

```sh
gutenmorgen Anna
gutenmorgen "Max Mustermann"
```

The command prints a randomized German good-morning greeting with emojis and a weekday-specific message.

## Installation

From this repository, link the package locally:

```sh
npm link
```

Then run:

```sh
gutenmorgen
```

You can also run it directly with Node:

```sh
node index.js
node index.js Anna
```

## Requirements

- Node.js
